# Explorer of the Family Roots

## What is this?

Explorer of the Family Roots is an interactive website that presents a structured and visual representation of a family. The site combines a traditional profile card layout with a visual family tree, allowing users to better understand relationships between family members.

Users can interact with the site by selecting a family member from the left-side navigation panel or clicking directly on a node within the family tree. When selected, the corresponding profile card expands while all others collapse, allowing the user to focus on one individual at a time. Each profile includes key information such as relationship, birth location, and important life events, along with links to view locations on a map.

## Why does it exist?

The purpose of this project is to present family information in a more meaningful and visual way than a traditional list or document. By combining interactivity with a structured layout, the site helps users understand relationships at a glance while still allowing them to explore individual details.

This project also serves as a demonstration of the web development process, showing how a basic layout can evolve into a more refined and user-friendly experience through iteration, design improvements, and responsiveness.

## What changed from Project 01 to Project 02?

The most significant change from Project 01 to Project 02 is the transition from a simple card-based layout to a structured and visual family tree.

In Project 01, the site displayed family members as individual profile cards with basic interaction. While functional, it did not clearly represent relationships between individuals.

For Project 02, several major improvements were made:

* **Added a visual family tree layout** that groups grandparents, parents, and children and visually connects them.
* **Implemented connector lines** to represent relationships between family members.
* **Integrated interactivity between components**, allowing users to select individuals from the tree, navigation menu, or cards, all synced together.
* **Improved responsive design**, with different layouts for desktop, tablet, and mobile to ensure usability across devices.
* **Added a background tree image** on desktop to enhance the visual theme without affecting readability.
* **Standardized content formatting**, including consistent naming structure and the use of b., m., and d. for life events.
* **Expanded and refined content**, improving clarity and organization of family information.

Overall, Project 02 represents a more complete and polished version of the original concept. The site is more visually meaningful, easier to navigate, and better communicates family relationships while maintaining a clean and responsive design.
